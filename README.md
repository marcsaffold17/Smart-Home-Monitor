# Smart Home Monitor
## Created for CS 4432: Sensors & IoT
- Security monitor developed using Raspberry Pi 5 that detects movement, distance, gas, and fire levels
- Designed for seniors who have experienced sensory loss
- This project was presented to a group of seniors on 12/3/25 and to the UMD CS department on 12/4/25
- Demonstration slideshow is attached as Monitor-Presentation.pdf
- Monitor feedback form attached as monitor_feedback_form.pdf
- Monitor code is attached as monitor_code.py (does not run without RPi and appropriate sensors)
